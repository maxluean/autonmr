 Solution
Phase 1 : Data & Reference Library                       
└─ สร้างพจนานุกรม 40+ สาร จาก HMDB+BMRB+NP-MRD       
                                                             
Phase 2 : Core Engine                                    
└─ Preprocessing + Peak Detection + Annotation            
    • Denoise, Baseline Correction, Normalize               
    • Auto Peak Detection                                   
    • Template Matching vs Reference Library                
    • MSI Confidence Levels (L1-L4)                         
                                                             
Phase 3 : ML/AI Training                                 
└─ Random Forest + 1D CNN Ensemble                        
    • Synthetic training data (2,000 samples)               
    • Multi-label classification                            
    • Cross-validation                                      
                                                             
Phase 4 : Demo UI (Streamlit)                            
└─ Web application for end-users                          
    • Upload NMR spectrum                                   
    • Auto annotation + confidence scoring                  
    • Export report (PDF/Excel)
